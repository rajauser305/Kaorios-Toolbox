# Kaorios Toolbox Guide

# Step 1:
 • Download and install the latest APK version.
# Step 2:
 • Copy the file "system/framework/framework.jar" to any folder in internal storage.
# Step 3: 
 • Open the KaoriosToolbox application and access FrameworkPatcher.
 
 • Select the copied framework file and click "Patch Framework".
 
 • Once successful, you can create a module or replace into the system (rw).
 
 • If it fails, check the framework to see if it contains hooks for custom ROMs; if so, you'll need to remove those hooks before using FrameworkPatcher.
# Step 4: 
 • If you want to create a module to flash via Root Manager, press Create Module.
 
 • In newer versions of KSU, KSUN, SUKISU, etc., you will need to flash the Hybrid Mount Meta Module before flashing the Kaoriostoolbox Module.
